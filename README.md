
# 💍 Atividade 2 - Introdução ao Flutter

Este repositório contém a resolução da segunda atividade prática de desenvolvimento mobile com Flutter. O objetivo do projeto é explorar os componentes estruturais básicos de um aplicativo e dominar a estilização de interfaces.

## 💡 O Conceito
Para ir além do básico exigido na atividade, o aplicativo foi estilizado com uma temática inspirada no universo de **O Senhor dos Anéis (J.R.R. Tolkien)**. A interface simula um artefato antigo e mágico, utilizando:
- Gradientes de fundo sombrios (remetendo a Moria).
- Cores hexadecimais precisas (Verde Élfico e Dourado do Um Anel).
- Fontes serifadas e cursivas para simular uma escrita antiga e encantada.
- Adaptação de um poema clássico da obra para o contexto de programação em Flutter.

## 🛠️ Tecnologias e Widgets Explorados
Durante o desenvolvimento, foram aplicados e consolidados os seguintes conceitos:
- `MaterialApp` e estruturação base.
- `Scaffold` e suas propriedades (`appBar`, `body`, `backgroundColor`).
- `Text` e `TextStyle` (fontFamily, fontWeight, letterSpacing, height, color).
- `Container` com `BoxDecoration` (LinearGradient, Border, BoxShadow).
- Estruturação de layout em eixos com `Column` e `Row`.
- Espaçamentos usando `SizedBox` e `Padding`.

## 🚀 Como executar o projeto

1. Certifique-se de ter o [Flutter SDK](https://docs.flutter.dev/get-started/install) instalado.
2. Clone este repositório:
   ```bash
   git clone https://github.com/agr3w/Att_2_Flutter_Desenvolvimento_Mobile_Unifacear.git
1.  Acesse a pasta do projeto e baixe as dependências:


    ```
    flutter pub get
    ```

2.  Conecte um emulador ou dispositivo físico e rode o app:


    ```
    flutter run
    ```
